# Burme SRT Pro 🇲🇲

> Myanmar Social Media Style Subtitle Generator

## Description

**Burme SRT Pro** is a social media style subtitle generator designed for Myanmar/Burmese content creators. Create beautiful SRT subtitles for your YouTube videos with Burmese + English support.

## Features

- 🎬 Fetch SRT from YouTube videos
- 📝 SRT Editor with syntax highlighting
- 📋 Copy to Clipboard
- ⬇️ Download SRT File
- 🎭 Animation Preview
- 🌐 Modern UI Design

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animation**: Three.js
- **Icons**: Font Awesome 6

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/burme-srt-pro.git

# Install dependencies
cd burme-srt-pro
npm install

# Run development server
npm run dev
```

## Deployment

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```
